# ⚽ Strikers FC — Team Management App

A full-stack web app built with **Python (Flask)** + **HTML/CSS/JS** to manage your football team.

## Features
- 📊 **Dashboard** — Season stats, top scorers, recent results
- 👥 **Squad Roster** — Player cards with goals, assists, appearances
- 🏟 **Matches** — Results, scores, filter by completed/upcoming
- 🏆 **Leaderboard** — Rankings by goals, assists, appearances
- 📋 **Attendance** — Track who showed up to each match
- 📅 **Upcoming Fixtures** — Countdown to next matches

## Setup

### Requirements
- Python 3.8+
- Flask

### Install & Run
```bash
# Install Flask
pip install flask

# Run the app
python app.py
```

Then open your browser at: **http://localhost:5050**

## Usage
- **Add players** from the Squad page
- **Log match results** from the Matches page
- **Mark attendance** after each game
- All data is saved to `data.json` automatically

## Tech Stack
- **Backend**: Python + Flask (REST API)
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Storage**: JSON file (data.json)
- **Fonts**: Bebas Neue + Barlow (Google Fonts)
